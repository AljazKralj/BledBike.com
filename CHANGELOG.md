# Changelog

All notable changes to the BledBike project.

## [0.2.0] - 2026-02-03

### Added
- **GDPR Cookie Consent Banner**
  - Slide-up banner on first visit
  - Accept/Decline options
  - Consent stored in localStorage
  - Clean, unobtrusive design matching site aesthetic
  
- **Privacy Policy Page** (`/privacy`)
  - Comprehensive GDPR-compliant privacy policy
  - Cookie usage explanation
  - User rights information
  - Contact details for privacy inquiries
  - Linked from cookie banner and footer

### Changed
- Footer now includes Privacy Policy link
- Layout updated to include cookie consent component

## [0.1.0] - 2026-02-03

### Added
- Initial Next.js 16 project setup with TypeScript
- Tailwind CSS 4 configuration
- Homepage with sections:
  - Hero section with gradient background
  - Service showcase (rentals & tours)
  - Pricing cards for 3 bike types
  - Guided tour offerings
  - "Why Choose Us" feature grid
  - Location and contact information
- Responsive navigation header (fixed)
- Footer with links and social media
- SEO-optimized metadata
- Mobile-first responsive design
- Project documentation:
  - README.md
  - PROJECT_PLAN.md
  - DEPLOY.md

### Tech Stack
- Next.js 16 (App Router)
- TypeScript 5
- Tailwind CSS 4
- React 19
